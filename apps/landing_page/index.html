Frequently asked questions

Does Flixor require a Plex Pass?

No, you can use your existing Plex server and account. Some premium features may require Plex Pass, just like in official apps.

How does playback work?

We stream via Plex using HLS/DASH with HLS.js/DASH.js in the browser, supporting DirectPlay where possible and transcode fallback via Plex.

Is this open source?

Yes. Clone, build, and deploy the static web app to any CDN or host that serves static files.
